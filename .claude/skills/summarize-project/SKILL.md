# Summarize Project

Get an overview of a project by examining its structure and key files.

## When to Use

- When starting work on an unfamiliar codebase
- When you need to understand how a project is organized

## What It Does

1. Explore directory structure
2. Read project documentation
3. Examine configuration files
4. Explore directory structure
5. Review source code
6. Explore directory structure

## Usage

```
/summarize-project
```

Or run the script directly on a specific directory:

```bash
bun .claude/skills/summarize-project/script.ts /path/to/project
```
